Szyfr Cezara

Szyfr Cezara to rodzaj szyfru podstawieniowego, w którym każda litera tekstu jawnego (niezaszyfrowanego) zastępowana jest inną literą (szyfr monoalfabetyczny) wyznaczoną poprzez stałe przesunięcie przy czym kierunek przesunięcia jest zachowany.

Poniżej przedstawiono funkcję odpowiadającą za roszyfrowywanie pliku z tekstem w języku angielskim, znajdując przesunięcie. 


void decryption(char message[], int n, int shift, char *filename) {

   char de_message[n];
   int i;
   char a;
   FILE* fptr;
   fptr = fopen(filename,"w");

   //dekodowanie
   for (i = 0; i < n; i++) {
      if (message[i] > ='a' && message[i] <= 'z') {
           //En(x) = (a - n) % 26 musi wyjsc zakres 0-25
           a = (message[i]-'a'-shift)%26;
           if(a < 0) a = a + 26;
           else if(a > 25) a=a-26;
           de_message[i] = a + 'a';
       } else if (message[i] >= 'A' && message[i] <= 'Z') {
           de_message[i] = ((message[i] - 'A' - shift) % 26 + 'A');
           a = (message[i] - 'A' - shift) % 26;
           if (a < 0) a = a + 26;
           else if (a > 25) a = a - 26;
           de_message[i] = a + 'A';
        } else {
           de_message[i] = message[i];
           continue;
        }
   }
   i = 0;
   while (i < n) {
       fprintf(fptr,"%c",de_message[i]);
       i++;
   }
}
