Model Isinga

Model Isinga to model matematyczny wykorzystywany w mechanice statystycznej do badań nad przejściami fazowymi.

W projekcie rozważamy model na dwuwymiarowej sitce kwadratowej z periodycznymi warunkami brzegowymi, otrzymując w ten sposób topologię torusa. Każdy węzeł sieci posiada spin, który może przyjąć wartość +1 lub -1.

Modele, którymi rządzą reguły dynamiczne, a nie jakieś przewidziane z góy zasady, można analaziować przy pomocy metod Monte Carlo. W tym przypadku zastosowano algorytm Metropolisa- Hastingsa. Poniżej przedstawiono kod odpowiadający jednemu krokowi Monte Carlo. 


for (int stp = 0; stp < MCS; stp++) {
    for (int i = 0; i < L; i++) {
        for (int j = 0; j < L; j++) {
            double dE = deltaE(i, j, iP, iN, lattice);
                if (dE <= 0) {
                    lattice[i][j] = -lattice[i][j];
                } else {
                    double rand = randDouble(mt_rand);
                    if (rand < exp(-dE / (T))) {
                        lattice[i][j] = -lattice[i][j];
                    }
                }
     }
}
